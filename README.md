# 📊 Business Insights Dashboard

A modern and responsive analytics dashboard built with **React**, **Vite**, and **Local Storage**. This project was developed as an internship demonstration to showcase frontend development skills, data visualization, authentication workflows, and dashboard design principles.

## 🚀 Features

- 🔐 User Registration and Login
- 👤 Local Storage-based Authentication
- 📈 Interactive Analytics Dashboard
- 📊 KPI Cards and Data Visualization
- 📝 CRUD Operations for Analytics Data
- 🔍 Search and Filtering Functionality
- 📱 Fully Responsive Design
- 🌙 Light and Dark Mode Support
- 🔔 User-Friendly Notifications
- ⚡ Fast Performance with Vite

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

### Data Management
- Browser Local Storage

### Visualization
- Chart.js / Recharts

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/SHIVAM200669/business-insights-dashboard.git
```

2. Navigate to the project directory:

```bash
cd business-insights-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```
